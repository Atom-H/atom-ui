# Radio

单选框, 支持v-model. 可单独使用, 也可配合cell组件使用

# 基本使用

{
    data(){
        return {select1: 1}
    },
}

    <a-group>
        <a-cell>
            <a-radio  v-model="select1" :value="1">选项1</a-radio>
        </a-cell>

        <a-cell>
            <a-radio v-model="select1" :value="2">选项2</a-radio>
        </a-cell>
    </a-group>

# API

# props
参数说明类型默认值可选值是否必选
value选项值, 选中后后会同步到v-model中Any-是
# slot
名称说明
default标题
# events
名称说明参数参数类型
change返回当前选中值当前选中值Any
